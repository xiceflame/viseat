## 🧠 核心技术亮点

### 亮点一：基线追踪法

**问题**：传统方案每次都重新识别食物，容易出错

**我们的方案**：
```
第一次拍照 → 记住所有食物（建立基线）
     ↓
后续拍照 → 只看还剩多少
     ↓
消耗量 = 开始时的量 - 现在剩的量
```

**好处**：
- ✅ 计算更准确
- ✅ 速度更快
- ✅ 不会搞混食物

---

### 亮点二：智能发现被遮挡的食物

**场景**：盖浇饭的米饭被菜盖住了

```
      开始时                    翻动后
   ┌──────────┐             ┌──────────┐
   │ 🥩 🥬 🥕 │             │ 🍚 🥩   │
   │    🥔    │    ──→      │  🥬 🥕  │
   │ ──────── │             │   🍚    │
   │  (米饭被  │             │ (米饭露 │
   │   遮住)   │             │  出来)  │
   └──────────┘             └──────────┘
   
   识别结果:                 识别结果:
   牛肉、蔬菜、土豆          发现新食物：米饭！
   355大卡                   更新总量：615大卡
```

**系统会自动**：
1. 发现米饭是新出现的
2. 提醒用户更新食物清单
3. 重新计算总热量
